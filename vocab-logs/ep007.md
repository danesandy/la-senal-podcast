# Vocabulario — El venezolano

## Nuevo

- **el ingeniero / la ingeniera** — engineer
- **arreglar** — to fix, repair
- **la herramienta** — tool
- **la empresa** — company, firm
- **instalar** — to install
- **el principio** — the beginning ('desde el principio')
- **el termo** — thermos
- **la arepa** — arepa (corn cake — contested territory)
- **tener razón** — to be right
- **chévere** — cool, great (Venezuelan slang)
